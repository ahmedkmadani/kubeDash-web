frequently asked queries

What is KubeDash?

KubeDash is a mobile app designed for iOS and Android that allows you to seamlessly view and manage your Kubernetes clusters. It provides a user-friendly interface to monitor cluster resources, deploy applications, and access vital information securely.

How can I access my Kubernetes cluster using KubeDash?

KubeDash supports seamless integration with your existing Kubernetes cluster. You can securely connect to your cluster by using your Kubernetes configuration file (kubeconfig). No direct access to your cluster is required, ensuring the security of your cluster while giving you full control from your mobile device.

What platforms is KubeDash available on?

Currently, KubeDash is available for both iOS and Android platforms.

Can I manage multiple clusters using KubeDash?

KubeDash allows you to focus on a single Kubernetes cluster at a time. You can securely configure and switch between different clusters using your kubeconfig file, ensuring a focused and secure management experience.
